\section*{ \fbox{Задание 1.} Расчет коэффициента усиления}
\begin{quote}
    \textit{Для каждого режима коррекции и для каждой частоты из протокола измерений (\cref{tab:afc_data}) рассчитать коэффициент усиления $K$ по напряжению.}
\end{quote}

Коэффициент усиления $K$ определяется как отношение выходного напряжения $U_\text{вых}$ к входному $U_\text{вх}$.
\begin{equation}
    K = \frac{U_\text{вых}}{U_\text{вх}}
    \label{eq:gain}
\end{equation}

В качестве примера приведем расчет для схемы без коррекции на частоте $f = 10$ кГц (данные из \cref{tab:afc_data}):
\begin{equation}
    K = \frac{\val[4]{p.afc.woc.outp[9]}}{\val[4]{p.afc.woc.inp[9]}} = \val[2]{r.afc.woc.K[9]}
\end{equation}

Аналогичные вычисления проводятся для всех остальных измерений. Сводные результаты расчетов представлены в \cref{tab:gain_results}.

\section*{ \fbox{Задание 3.} Определение граничных частот}
\begin{quote}
    \textit{Определить нижнюю и верхнюю граничные частоты ($f_\text{н.гр}$ и $f_\text{в.гр}$) для каждой схемы, исходя из условия $K(f_\text{гр}) \approx 0.707 \cdot K_\text{max}$.}
\end{quote}

Граничные частоты определяются на уровне, где коэффициент усиления снижается до $1/\sqrt{2} \approx 0.707$ от своего максимального значения в полосе пропускания.
\begin{equation}
    K_\text{гр} = \frac{K_\text{max}}{\sqrt{2}}
    \label{eq:cutoff_level}
\end{equation}

Для схемы без коррекции максимальный коэффициент усиления $K_\text{max} = \val[2]{r.afc.woc.K_max}$. Тогда уровень среза составляет:
\begin{equation}
    K_\text{гр} = \frac{\val[2]{r.afc.woc.K_max}}{\sqrt{2}} = \val[2]{r.afc.woc.K_cutoff}
\end{equation}

Граничные частоты определяются на уровне $K_\text{гр} = K_\text{max} / \sqrt{2}$. Для их нахождения используется метод логарифмической интерполяции между двумя ближайшими экспериментальными точками $(f_1, K_1)$ и $(f_2, K_2)$, которые <<окружают>> искомый уровень $K_\text{гр}$. Итоговые значения сведены в \cref{tab:cutoff_freqs}.

\section*{ \fbox{Задание 4.} Расчет параметров искажений импульсов}
\begin{quote}
    \textit{Рассчитать значения длительности фронта $\tau_\text{ф}$ и спада вершины $\delta = \Delta U / U_m$ (\%), исходя из измеренных граничных частот.}
\end{quote}

Искажения формы прямоугольного импульса напрямую связаны с граничными частотами усилителя.

\subsection*{Длительность фронта}
Длительность фронта $\tau_\text{ф}$ обратно пропорциональна верхней граничной частоте $f_\text{в.гр}$:
\begin{equation}
    \tau_\text{ф} = \frac{0.35}{f_\text{в.гр}}
    \label{eq:rise_time}
\end{equation}
Для схемы без коррекции:
\begin{equation}
    \tau_\text{ф} = \frac{0.35}{\val[0]{r.afc.woc.f_high}} = \val[2]{r.pulse.woc_tau_f * 1e6} \text{ мкс}
\end{equation}

\subsection*{Спад вершины}
Приближенная формула $\delta = 628 \cdot t_\text{и} \cdot f_\text{н.гр}$ применима только для малых искажений. В общем случае следует использовать точную формулу, описывающую экспоненциальный спад напряжения на выходе RC-фильтра высоких частот:
\begin{equation}
    \delta [\%] = \left(1 - e^{-t_\text{и} / \tau_\text{н}}\right) \cdot 100\%, \quad \text{где } \tau_\text{н} = \frac{1}{2\pi f_\text{н.гр}}
\end{equation}
Для схемы без коррекции при $t_\text{и} = T/2 = \val[1]{p.pulse.nf / 2 * 1000}$ мс и $f_\text{н.гр} = \val[1]{r.afc.woc.f_low}$ Гц:
\begin{equation}
    \delta = \left(1 - \exp\left(-\frac{\val[3]{p.pulse.nf / 2}}{1 / (2\pi \cdot \val[1]{r.afc.woc.f_low})}\right)\right) \cdot 100\% = \val[2]{r.pulse.woc_delta} \%
\end{equation}

\textit{Анализ:} 
1. Высокое расчетное значение $\delta = \val[1]{r.pulse.woc.delta}\%$ для режима без коррекции обусловлено высокой частотой среза $f_{\text{н.гр}} \approx 84\text{ Гц}$ в протоколе. На осциллограмме спад значителен (около $1/3$ амплитуды), что подтверждает порядок величины.
2. В режиме НЧК наблюдается компенсация спада ($\delta = \val[1]{r.pulse.nck.delta}\%$), что соответствует плоской вершине на осциллограмме.
3. ИВЧК дает минимальный $\tau_{\phi}$, но сопровождается значительным выбросом (overshoot), видимым на фото.
