# О лабах и защите
- Всего будет 5 лаб
- Защиты нет, только снятие и обработка
- Следующую для снятия лабу сообщают в конце крайнего занятия
- Подготовка не нужна, но понимать что делать - обязательно
- Работы длинные. Вопросы по методе можно спокойно задаватьэ
- Протокол - чистый лист + №группы + ФИО + ID Работы. Заполнять в свободном формате.
- Могут попросить что-то перемерить, поэтому оставляем резерв по времени на паре
- Сдавать лабы можно в любое удобное время, они могут не с первого раза пройти (если что-то будет неверно).
- Один отчет на всю команду, но можно и индивидуально

# Требования к отчету
- Титульник
- Протокол
- Схемы
- Результаты измерений в виде таблицы, графиков, осциллограм
  - Для АЧХ - semilogx
  - Для АХ - линейная
  - Для сравнений кривых - на одном графике (должны быть различимы в ч/б)
  - Обозначение осей и размерностей
  - Не путать осциллограмы!

# ВАЖНО
- В лабах 5 и 10 обозначения методы не совпадают с обозначениями на установке
