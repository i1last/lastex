\section{Расчет частотных характеристик цепи}

Амплитудно-фазовая характеристика (АФХ) цепи получается путем замены оператора $s$ на $j\omega$ в выражении передаточной функции $H(s)$ (см. \cref{form:hs}). Однако прежде, зная нули и полюсы (см. \cref{form:zeros,form:poles}), разложим ее на множители:
\begin{equation}
    H(s) = \frac{\pv[.2]{tf['K']}(s - j\pv[.2]{tf['z1_im']})(s + j\pv[.2]{tf['z1_im']})}{(s + \pv[.2]{fact['alpha1']})(s + \pv[.2]{fact['alpha2']} - j\pv[.2]{fact['beta2']})(s + \pv[.2]{fact['alpha2']} + j\pv[.2]{fact['beta2']})}
    \label{form:hs_simply}
\end{equation}

Подставим $s = j\omega$:
\begin{equation}
    H(j\omega) = \frac{
        \pv[.2]{tf['K']}
        (j\omega - j\pv[.2]{tf['z1_im']})
        (j\omega + j\pv[.2]{tf['z1_im']})
    }{
        (j\omega + \pv[.2]{fact['alpha1']})
        (j\omega + \pv[.2]{fact['alpha2']} - j\pv[.2]{fact['beta2']})
        (j\omega + \pv[.2]{fact['alpha2']} + j\pv[.2]{fact['beta2']})
    }
\end{equation}

\subsection{Амплитудно- и фазо-частотная характеристики}

Амплитудно-частотная характеристика (АЧХ) определяется как модуль АФХ:
\begin{equation}
    A(\omega) = |H(j\omega)| = 
    \frac{
        \pv[.2]{tf['K']}
        \left|\omega^2 - \pv[.2]{tf['z1_im']}^2\right|
    }{
        \sqrt{\omega^2 + \pv[.2]{fact['alpha1']}^2}
        \sqrt{(\omega - \pv[.2]{fact['beta2']})^2 + \pv[.2]{fact['alpha2']}^2}
        \sqrt{(\omega + \pv[.2]{fact['beta2']})^2 + \pv[.2]{fact['alpha2']}^2}
    }
    \label{form:AChH}
\end{equation}

Фазо-частотная характеристика (ФЧХ) определяется как аргумент АФХ:
\begin{equation}
    \begin{multlined}
        \Phi(\omega) = \arg\left[H(j\omega)\right] =
          \arg{( \pv[.2]{tf['z1_im']}^2 - \omega^2 )}
        - \arctan{ \frac{ \omega }{ \pv[.2]{fact['alpha1']} } } - \\
        - \arctan{ \frac{ \omega + \pv[.2]{fact['beta2']} }{ \pv[.2]{fact['alpha2']} } }
        - \arctan{ \frac{ \omega - \pv[.2]{fact['beta2']} }{ \pv[.2]{fact['alpha2']} } },
    \end{multlined}
    \label{form:FChH}
\end{equation}
где $\arg{( \pv[.2]{tf['z1_im']}^2 - \omega^2 )} = \begin{cases} 0^\circ, & \omega \le \pv[.2]{tf['z1_im']} \\ -180^\circ, & \omega > \pv[.2]{tf['z1_im']} \end{cases}$.

\subsection{Анализ характеристик в контрольных точках}

Проверка значений АФХ на граничных частотах (по \cref{form:hs}):
\begin{itemize}
    \item При $\omega = 0 ~ (s \to 0)$: $H(s=0) = \frac{\pv[.3]{tf['N0']}}{\pv[.3]{tf['D0']}} = \pv[.2]{freq['A0']}$.
    \item При $\omega \to \infty ~ (s\to\infty)$: $H(s\to\infty) \sim {s^2}/{s^3} \to 0$.
    \item На частоте нуля $\omega_0 = \pv[.2]{tf['z1_im']} ~ (s = j\pv[.2]{tf['z1_im']})$: $H(s = j\pv[.2]{tf['z1_im']}) = 0$.
\end{itemize}

\subsection{Полоса пропускания и время запаздывания}

Полоса пропускания определяется по уровню $0.707 \cdot A(0) = \pv[.3]{freq['level']}$. Исходя из графиков АЧХ (\cref{fig:afc}), полоса пропускания составляет 
\begin{equation}
    \Delta\omega \in [0; \pv[.2]{freq['w_c']}].
    \label{form:omega_propuskaniya}
\end{equation}
 
Время запаздывания сигнала при прохождении через цепь в области низких частот определяется как
\begin{equation}
    t_{\text{з}} =
    \frac{|d\Phi|}{d\omega} =
    \frac{
        \operatorname{rad}(|\pv[.1]{freq['td']['deg']}^\circ|)
    }{
        \pv[.1]{freq['td']['w']}
    }
    \approx
    \pv[.2]{freq['td']['res']}.
\end{equation}
