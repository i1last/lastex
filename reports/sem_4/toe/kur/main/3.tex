\section{Расчет частотных характеристик цепи}

Амплитудно-фазовая характеристика (АФХ) цепи получается путем замены оператора $s$ на $j\omega$ в выражении передаточной функции $H(s)$ (см. \cref{form:hs}). Однако прежде, зная нули и полюсы (см. \cref{form:zeros,form:poles}), разложим ее на множители:
\begin{equation}
    H(s) = \frac{(s - j\val[2]{tf.z1_im})(s + j\val[2]{tf.z1_im})}{(s + \val[2]{fact.alpha1})(s + \val[2]{fact.alpha2} - j\val[2]{fact.beta2})(s + \val[2]{fact.alpha2} + j\val[2]{fact.beta2})}
    \label{form:hs_simply}
\end{equation}

Подставим $s = j\omega$:
\begin{equation}
    H(j\omega) = \frac{
        (j\omega - j\val[2]{tf.z1_im})
        (j\omega + j\val[2]{tf.z1_im})
    }{
        (j\omega + \val[2]{fact.alpha1})
        (j\omega + \val[2]{fact.alpha2} - j\val[2]{fact.beta2})
        (j\omega + \val[2]{fact.alpha2} + j\val[2]{fact.beta2})
    }
\end{equation}

\subsection{Амплитудно- и фазо-частотная характеристики}

Амплитудно-частотная характеристика (АЧХ) определяется как модуль АФХ:
\begin{equation}
    A(\omega) = |H(j\omega)| = 
    \frac{
        \omega^2 + \val[2]{tf.z1_im}^2
    }{
        \sqrt{\omega^2 + \val[2]{fact.alpha1}^2}
        \sqrt{(\omega - \val[2]{fact.beta2})^2 + \val[2]{fact.alpha2}^2}
        \sqrt{(\omega + \val[2]{fact.beta2})^2 + \val[2]{fact.alpha2}^2}
        }
\end{equation}

Фазо-частотная характеристика (ФЧХ) определяется как аргумент АФХ:
\begin{equation}
    \begin{multlined}
        \Phi(\omega) = \arg\left[H(j\omega)\right] =
          \arctan{( \omega - \val[2]{tf.z1_im} )}
        + \arctan{( \omega + \val[2]{tf.z1_im} )} - \\
        - \arctan{ \frac{ \omega }{ \val[2]{fact.alpha1} } }
        - \arctan{ \frac{ \omega + \val[2]{fact.beta2} }{ \val[2]{fact.alpha2} } }
        - \arctan{ \frac{ \omega - \val[2]{fact.beta2} }{ \val[2]{fact.alpha2} } }
    \end{multlined}
    % \arg\{\val[3]{tf.N0} - \val[3]{tf.N2}\omega^2\} - \operatorname{arctg}\left(\frac{\val[3]{tf.D1}\omega - \val[3]{tf.D3}\omega^3}{\val[3]{tf.D0} - \val[3]{tf.D2}\omega^2}\right)
\end{equation}

\subsection{Анализ характеристик в контрольных точках}

Проверка значений АФХ на граничных частотах (по \cref{form:hs}):
\begin{itemize}
    \item При $\omega = 0 ~ (s \to 0)$: $H(s=0) = \frac{\val[3]{tf.N0}}{\val[3]{tf.D0}} = \val[2]{freq.A0}$.
    \item При $\omega \to \infty ~ (s\to\infty)$: $H(s\to\infty) \sim {s^2}/{s^3} \to 0$.
    \item На частоте нуля $\omega_0 = \val[2]{tf.z1_im} ~ (s = j\val[2]{tf.z1_im})$: $H(s = j\val[2]{tf.z1_im}) = 0$.
\end{itemize}

\subsection{Полоса пропускания и время запаздывания}

Полоса пропускания определяется по уровню $0.707 \cdot A(0) = \val[3]{freq.level}$. Исходя из графиков АЧХ (\cref{fig:afc}), полоса пропускания составляет $\Delta\omega \in [0; \val[2]{freq.w_c}]$.
