\section{Определение спектральных характеристик одиночного импульса воздействия}
\label{sec:oprspechar}
С учетом полученного ранее изображения по Лапласу, комплексный спектр импульсного воздействия определяется подстановкой $s = j\omega$:
\begin{equation}
    \begin{aligned}
        I_1(j\omega)
            &= I_1(s) \Big|_{s=j\omega} \\
            &= \frac{I_m}{j\omega} \left(
                1 - 2e^{-j\omega \frac12 t_{\text{и}}} + e^{-j\omega t_{\text{и}}}
            \right) = \\
            &=
                \frac{I_m}{j\omega} \left(
                    1 - e^{-j\omega\frac{t_\text{и}}2}
                \right)^2 = \\
            &=
                \frac{I_m}{j\omega} \left[
                    e^{-j\omega\frac{t_\text{и}}4} \cdot \left(
                        e^{j\omega t_\text{и} / 4} -
                        e^{-j\omega t_\text{и} / 4}
                    \right)
                \right]^2 = \\
            &=
                \frac{I_m}{j\omega} \left[
                    e^{-j\omega\frac{t_\text{и}}4} \cdot \left(
                        2j\sin(\omega t_\text{и} / 4)
                    \right)
                \right]^2 = \\
            &= \frac{I_m}{j\omega} e^{-j\omega\frac{t_\text{и}}4} \cdot (-4) \cdot \sin^2(\omega t_\text{и} / 4) = \\
            &= 4\sin^2(\omega t_\text{и} / 4) \cdot \frac{I_m}{\omega} e^{-j(\omega t_\text{и}/2 - 90^\circ)} = \\
            &= \frac{4I_m}{\omega} \sin^2(\omega t_\text{и} / 4) \cdot e^{j(90^\circ - \omega t_\text{и}/2)} = \\
            &= A_1(\omega)\cdot e^{j\Phi_1(\omega)}
    \end{aligned}
    \label{form:I1jw}
\end{equation}

Амплитудный спектр входного сигнала:
\begin{equation}
    A_1(\omega) = \left|
        \frac{4I_m}{\omega} \sin^2\left(\frac{\omega t_{\text{и}}}{4}\right)
    \right|
\end{equation}
 
Фазовый спектр входного сигнала:
\begin{equation}
    \Phi_1(\omega) = 90^\circ - \frac{\omega t_\text{и}}{2}
\end{equation}

Для построения графика амплитудного спектра определяются его нули, то есть значения частот $\omega_\text{yзл}^k$, при которых спектр равен нулю:
\begin{equation}
    A_1(\omega) = 0 \text{ при } \sin^2\left(\frac{\omega t_{\text{и}}}{4}\right) = 0
    \iff
    \frac{\omega t_{\text{и}}}{4} = \pi k
    \implies
    \omega_\text{yзл}^k = \frac{4\pi k}{t_\text{и}}
\end{equation}

Частоты первых нулей для заданных длительностей импульсов:
\begin{itemize}
    \item Для $t_{\text{и1}}$: $\omega_{\text{yзл1}}^1 \approx \pv[.3]{ex7['ti1']['w_y1']}$
    \item Для $t_{\text{и2}}$: $\omega_{\text{yзл2}}^1 \approx \pv[.3]{ex7['ti2']['w_y1']}$
\end{itemize}

При вычислении значения амплитудного спектра на нулевой частоте возникает неопределенность вида $\left[ \frac{0}{0} \right]$. Применим первый замечательный предел:
\begin{equation}
    \begin{aligned}
        A_1(0) 
        &= \lim_{\omega \to 0} 4\frac{I_m}{\omega} \sin^2\left(
            \frac{\omega t_{\text{и}}}{4}
        \right) = \\
        &= 4I_m \lim_{\omega \to 0} \frac{
                \sin^2\left(\frac{\omega t_{\text{и}}}{4}\right)
        }{
                \omega
        }
        = \bcancel{4}I_m \lim_{\omega \to 0} \frac{
                \omega^{\cancel{2}} t^2_\text{и}
        }{
                \bcancel{4}\cdot 4 \cancel{\omega}
        } \\
        &= I_m \lim_{\omega \to 0} \frac{
                \omega t^2_\text{и}
        }{
                4
        }
        = 0
    \end{aligned}
\end{equation}

Контроль: значение амплитудного спектра на нулевой частоте равно площади входного импульса тока.
\begin{itemize}
    \item Для $t_{\text{и1}}$: $A_1(0) = 0$
    \item Для $t_{\text{и2}}$: $A_1(0) = 0$
\end{itemize}

\end{itemize}
