\section{Определение спектральных характеристик одиночного импульса воздействия}
\label{sec:oprspechar}
С учетом полученного ранее изображения по Лапласу, комплексный спектр импульсного воздействия определяется подстановкой $s = j\omega$:
\begin{equation}
    \begin{aligned}
        I_1(j\omega)
            &= I_1(s) \Big|_{s=j\omega} \\
            &= \frac{I_m}{j\omega} \left(
                1 - 2e^{-j\omega \frac14 T} + e^{-j\omega \frac12 T}
            \right) = \\
            &=
                \frac{I_m}{j\omega} \left(
                    1 - e^{-j\omega\frac{T}4}
                \right)^2 = \\
            &=
                \frac{I_m}{j\omega} \left[
                    e^{-j\omega\frac{T}8} \cdot \left(
                        e^{j\omega T / 8} -
                        e^{-j\omega T / 8}
                    \right)
                \right]^2 = \\
            &=
                \frac{I_m}{j\omega} \left[
                    e^{-j\omega\frac{T}8} \cdot \left(
                        2j\sin(\omega T / 8)
                    \right)
                \right]^2 = \\
            &= \frac{I_m}{j\omega} e^{-j\omega\frac{T}4} \cdot (-4) \cdot \sin^2(\omega T / 8) = \\
            &= 4\sin^2(\omega T / 8) \cdot \frac{I_m}{\omega} e^{-j(\omega T/4 - 90^\circ)} = \\
            &= \frac{4I_m}{\omega} \sin^2(\omega T / 8) \cdot e^{j(90^\circ - \omega T/4)} = \\
            &= A_1(\omega)\cdot e^{j\Phi_1(\omega)}
    \end{aligned}
    \label{form:I1jw}
\end{equation}

Амплитудный спектр входного сигнала:
\begin{equation}
    A_1(\omega) = \left|
        \frac{4I_m}{\omega} \sin^2\left(\frac{\omega T}{8}\right)
    \right|
\end{equation}
 
Фазовый спектр входного сигнала:
\begin{equation}
    \Phi_1(\omega) = 90^\circ - \frac{\omega T}{4}
\end{equation}

Для построения графика амплитудного спектра определяются его нули, то есть значения частот $\omega_\text{yзл}^k$, при которых спектр равен нулю:
\begin{equation}
    A_1(\omega) = 0 \text{ при } \sin^2\left(\frac{\omega T}{8}\right) = 0
    \iff
    \frac{\omega T}{8} = \pi k
    \implies
    \omega_\text{yзл}^k = \frac{8\pi k}{T}
\end{equation}

Частоты первых нулей для заданных длительностей импульсов:
\begin{itemize}
    \item Для $T_1 = 2t_{\text{и1}}$: $\omega_{\text{yзл1}}^1 \approx \pv[.3]{ex7['ti1']['w_y1']}$
    \item Для $T_2 = 2t_{\text{и2}}$: $\omega_{\text{yзл2}}^1 \approx \pv[.3]{ex7['ti2']['w_y1']}$
\end{itemize}

При вычислении значения амплитудного спектра на нулевой частоте возникает неопределенность вида $\left[ \frac{0}{0} \right]$. Применим первый замечательный предел:
\begin{equation}
    \begin{aligned}
        A_1(0) 
        &= \lim_{\omega \to 0} 4\frac{I_m}{\omega} \sin^2\left(
            \frac{\omega T}{4}
        \right) = \\
        &= 4I_m \lim_{\omega \to 0} \frac{
                \sin^2\left(\frac{\omega T}{8}\right)
        }{
                \omega
        }
        = 4I_m \lim_{\omega \to 0} \frac{
                \omega^{\cancel{2}} T^2
        }{
                64 \cancel{\omega}
        } \\
        &= I_m \lim_{\omega \to 0} \frac{
                \omega T^2
        }{
                16
        }
        = 0
    \end{aligned}
\end{equation}

Контроль: значение амплитудного спектра на нулевой частоте равно площади входного импульса тока.
\begin{itemize}
    \item Для $T_1 = 2t_{\text{и1}}$: $A_1(0) = 0$;
    \item Для $T_2 = 2t_{\text{и2}}$: $A_1(0) = 0$.
\end{itemize}
Площадь входного импульса равна нулю, поскольку это меандр, а значит проверка пройдена.

Ширина спектра определяется по 10\%-му амплитудному критерию. Из \cref{fig:plot_spec_A_ti1,fig:plot_spec_A_ti2} видно, что
\begin{equation}
    \Delta\omega_{\text{сп1}} \approx \pv[.2]{ex7['ti1']['w_sp']} \text{ при } t_\text{и} = \pv[.1]{n['ti1']},
\end{equation}
\begin{equation}
    \Delta\omega_{\text{сп2}} \approx \pv[.2]{ex7['ti2']['w_sp']} \text{ при } t_\text{и} = \pv[.1]{n['ti2']}.
\end{equation}

Сравнение ширины спектра воздействия и полосы пропускания цепи $\omega_{\text{ср}} \approx \pv[.2]{ex7['w_c']}$ (см. \cref{form:omega_propuskaniya}) показывает:
\begin{itemize}
    \item Для $t_{\text{и1}}$: $\Delta\omega_{\text{сп1}} \approx \omega_{\text{ср}}$.
        Таким образом $\Delta\omega_{\text{сп1}} \in (0;\,\pv[.2]{ex7['ti1']['w_sp']})$, а $\omega_{\text{ср}} \in (0;\,\pv[.2]{ex7['w_c']})$.

        Большая часть спектра укладывается в полосу пропускания цепи, следовательно, искажения выходного сигнала будут минимальны.

    \item Для $t_{\text{и2}}$: $\Delta\omega_{\text{сп2}} > \omega_{\text{ср}}$.
        Таким образом $\Delta\omega_{\text{сп2}} \in (0;\,\pv[.2]{ex7['ti2']['w_sp']})$, а $\omega_{\text{ср}} \in (0;\,\pv[.2]{ex7['w_c']})$.

        Большая часть спектра не укладывается в полосу пропускания цепи, попадая в полосу задерживания вида интегрирования, следовательно, искажения выходного сигнала будут большие.
\end{itemize}

Значение АС $A_1(0) = 0 \implies$ площадь входного сигнала равна $0$.
АЧХ $A(0)=0.5 \implies$ площадь выходного сигнала уменьшится в $2$ раза.
$A(\infty)=0 \implies$ скачок входного сигнала на выход не пройдёт.
