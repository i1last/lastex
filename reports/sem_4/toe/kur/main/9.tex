\section{Приближенный расчет реакции по спектру}
\label{sec:spectr_reaction}

Связь между спектральной плотностью сигнала ${I}_{2}(j\omega)$ \cref{form:I2spectr} и его временным представлением $i_2(t)$ устанавливается с помощью обратного преобразования Фурье:
\begin{equation}
    i_2(t) = \frac{1}{\pi} \int_{0}^{\infty} A_2(\omega) \cos(\omega t + \Phi_2(\omega)) d\omega,
    \label{form:i2tapprox}
\end{equation}
где $A_2(\omega)$ представлен в \cref{form:A2w}; $\Phi_2(\omega)$ представлен в \cref{form:Phi2w}.

Аналитическое вычисление данного интеграла затруднительно. Приближенный расчет осуществляется путем замены бесконечного верхнего предела интегрирования на конечное значение $\omega_{\text{max}}$ и применения методов численного интегрирования. 

Примем $\omega_{\text{max}} = 100 / t_и$, а для интегрирования будем использовать численный метод трапеций (реализованный библиотекой NumPy для языка программирования Python под названием trapz).

Продемонстрируем вычисление для двух длительностей меандра в контрольных точках для приближенного \cref{form:i2tapprox} и точного \cref{form:i2torig} решений:
\begin{itemize}
    \item Для $t_{\text{и}1} = \pv[.2]{n['ti1']}$ в момент $t = \pv[.2]{n['ti1']*t_offset_for_demo}$:
    \begin{align*}
        i_{2\text{ (прибл)}} = \pv[.4]{ex9['i2_approx_demo1']} \qquad
        i_{2\text{ (точн)}} = \pv[.4]{ex9['i2_exact_demo1']} 
    \end{align*}
    \item Для $t_{\text{и}2} = \pv[.2]{n['ti2']}$ в момент $t = \pv[.2]{n['ti2']*t_offset_for_demo}$:
    \begin{align*}
        i_{2\text{ (прибл)}} = \pv[.4]{ex9['i2_approx_demo2']} \qquad
        i_{2\text{ (точн)}} = \pv[.4]{ex9['i2_exact_demo2']} 
    \end{align*}
\end{itemize}
